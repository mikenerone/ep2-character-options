# *Eclipse Phase 2e Character Options* compendium content for Bubz's EP2 Foundry VTT system

This Foundry VTT module provides compendium content from the *Character Options* splatbook for Posthuman Studios'
tabletop role playing game, *Eclipse Phase* Second Edition (EP2). This content is for use with Bubz's [EP2 game
system][bubz-ep2e] for Foundry VTT.

This content is available to you thanks to the generosity of Posthuman Studios, who have graciously provided it under
the permissive CC BY-NC-SA license. Support them and the rest of these folks via the links below!

All creative credit goes to Posthuman Studios, and credit to @BigMistacke on Discord for a large part of the data
transcription.

### @BigMistacke on Discord
I know practically nothing about this helpful person on Discord who initially transcribed most of this data from
*Character Options* and shared it on Bubz's Discord server. @BigMistacke stated the intent to make a release after some
feedback, but has since left the server, and it seems likely that plans have changed. Sadly, attempts to communicate
have not been fruitful. In any case, @BigMistacke's hard work in repackaging Posthuman Studio's content in Foundry's
format is greatly appreciated.
